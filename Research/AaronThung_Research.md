## User Stories

- As a software engineer, I want the system to automatically pull my GitHub PR activity s**o that I** can provide accurate status updates without manual typing.
- **As a** project lead, **I want** an AI agent that monitors team presence (Active/Focus/DND) and scans linked calendars for free slots, s**o that** I can schedule asynchronous check-ins or live meetings without interrupting a developer's deep-work "Focus" state.
- As a team member, I want to log my daily mood and energy levels, so that I can stay connected to my own and the team’s well-being
